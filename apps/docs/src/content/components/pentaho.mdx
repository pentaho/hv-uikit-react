# Pentaho

The pentaho package contains a set of UI components designed to address the unique needs of Pentaho products,
that don't quite fit into the [core package](.) guidelines.

## Usage

```sh
npm install @hitachivantara/uikit-react-pentaho
```

```tsx
import { HvCanvasSidePanel } from "@hitachivantara/uikit-react-pentaho";
```
