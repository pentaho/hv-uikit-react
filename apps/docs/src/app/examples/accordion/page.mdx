### External container

Accordion sections managed outside the container. Only one section can be expanded at a time using externally controlled state, with optional header actions.

<CodeBlock code={externalContainerCode} />

### Internal container

Accordion sections nested inside an `HvSection` container. Collapsed sections are stacked inside the parent, forming a compact grouped layout.

<CodeBlock code={internalContainerCode} />

### Internal container with status

Accordion sections inside a parent `HvSection`, each with a status indicator and header actions to convey contextual information alongside the collapsible content.
